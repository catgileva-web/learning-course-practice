## Components

- [App Bar React component](docs/mui_docs/react-app-bar.md): The App Bar displays information and actions relating to the current screen.
- [Backdrop React Component](docs/mui_docs/react-backdrop.md): The Backdrop component narrows the user's focus to a particular element on the screen.
- [Bottom Navigation React component](docs/mui_docs/react-bottom-navigation.md): The Bottom Navigation bar allows movement between primary destinations in an app.
- [Circular, Linear progress React components](docs/mui_docs/react-progress.md): Progress indicators commonly known as spinners, express an unspecified wait time or display the length of a process.
- [CSS Baseline](docs/mui_docs/react-css-baseline.md): The CssBaseline component helps to kickstart an elegant, consistent, and simple baseline to build upon.
- [Detect click outside React component](docs/mui_docs/react-click-away-listener.md): The Click-Away Listener component detects when a click event happens outside of its child element.

- [Image List React component](docs/mui_docs/react-image-list.md): The Image List displays a collection of images in an organized grid.
- [InitColorSchemeScript component](docs/mui_docs/react-init-color-scheme-script.md): The InitColorSchemeScript component eliminates dark mode flickering in server-side-rendered applications.
- [Links](docs/mui_docs/react-link.md): The Link component allows you to easily customize anchor elements with your theme colors and typography styles.
- [No SSR React component](docs/mui_docs/react-no-ssr.md): The No-SSR component defers the rendering of children components from the server to the client.
- [Number field React component](docs/mui_docs/react-number-field.md): A React component for capturing numeric input from users.
- [React Accordion component](docs/mui_docs/react-accordion.md): The Accordion component lets users show and hide sections of related content on a page.
- [React Alert component](docs/mui_docs/react-alert.md): Alerts display brief messages for the user without interrupting their use of the app.
- [React Autocomplete component](docs/mui_docs/react-autocomplete.md): The autocomplete is a normal text input enhanced by a panel of suggested options.
- [React Avatar component](docs/mui_docs/react-avatar.md): Avatars are found throughout material design with uses in everything from tables to dialog menus.
- [React Badge component](docs/mui_docs/react-badge.md): Badge generates a small badge to the top-right of its child(ren).
- [React Box](docs/mui_docs/react-box.md): The Box component is a generic, theme-aware container with access to CSS utilities from MUI System.
- [React Breadcrumbs component](docs/mui_docs/react-breadcrumbs.md): A breadcrumbs is a list of links that help visualize a page's location within a site's hierarchical structure, it allows navigation up to any of the ancestors.
- [React Button component](docs/mui_docs/react-button.md): Buttons allow users to take actions, and make choices, with a single tap.
- [React Button Group component](docs/mui_docs/react-button-group.md): The ButtonGroup component can be used to group related buttons.
- [React Card component](docs/mui_docs/react-card.md): Cards contain content and actions about a single subject.
- [React Checkbox component](docs/mui_docs/react-checkbox.md): Checkboxes allow the user to select one or more items from a set.
- [React Chip component](docs/mui_docs/react-chip.md): Chips are compact elements that represent an input, attribute, or action.
- [React Container component](docs/mui_docs/react-container.md): The container centers your content horizontally. It's the most basic layout element.
- [React Dialog component](docs/mui_docs/react-dialog.md): Dialogs inform users about a task and can contain critical information, require decisions, or involve multiple tasks.
- [React Divider component](docs/mui_docs/react-divider.md): The Divider component provides a thin, unobtrusive line for grouping elements to reinforce visual hierarchy.
- [React Drawer component](docs/mui_docs/react-drawer.md): The navigation drawers (or "sidebars") provide ergonomic access to destinations in a site or app functionality such as switching accounts.
- [React Floating Action Button (FAB) component](docs/mui_docs/react-floating-action-button.md): A Floating Action Button (FAB) performs the primary, or most common, action on a screen.
- [React Grid component](docs/mui_docs/react-grid.md): The responsive layout grid adapts to screen size and orientation, ensuring consistency across layouts.
- [React GridLegacy component](docs/mui_docs/react-grid-legacy.md): The Material Design responsive layout grid adapts to screen size and orientation, ensuring consistency across layouts.
- [React Icon Component](docs/mui_docs/icons.md): Guidance and suggestions for using icons with Material UI.
- [React List component](docs/mui_docs/react-list.md): Lists are continuous, vertical indexes of text or images.
- [React Masonry component](docs/mui_docs/react-masonry.md): Masonry lays out contents of varying dimensions as blocks of the same width and different height with configurable gaps.
- [React Menu component](docs/mui_docs/react-menu.md): Menus display a list of choices on temporary surfaces.
- [React Modal component](docs/mui_docs/react-modal.md): The modal component provides a solid foundation for creating dialogs, popovers, lightboxes, or whatever else.
- [React Pagination component](docs/mui_docs/react-pagination.md): The Pagination component enables the user to select a specific page from a range of pages.
- [React Paper component](docs/mui_docs/react-paper.md): The Paper component is a container for displaying content on an elevated surface.
- [React Popover component](docs/mui_docs/react-popover.md): A Popover can be used to display some content on top of another.
- [React Popper component](docs/mui_docs/react-popper.md): A Popper can be used to display some content on top of another. It's an alternative to react-popper.
- [React Portal component](docs/mui_docs/react-portal.md): The Portal component lets you render its children into a DOM node that exists outside of the Portal's own DOM hierarchy.
- [React Radio Group component](docs/mui_docs/react-radio-button.md): The Radio Group allows the user to select one option from a set.
- [React Rating component](docs/mui_docs/react-rating.md): Ratings provide insight regarding others' opinions and experiences, and can allow the user to submit a rating of their own.
- [React Select component](docs/mui_docs/react-select.md): Select components are used for collecting user provided information from a list of options.
- [React Skeleton component](docs/mui_docs/react-skeleton.md): Display a placeholder preview of your content before the data gets loaded to reduce load-time frustration.
- [React Slider component](docs/mui_docs/react-slider.md): Sliders allow users to make selections from a range of values.
- [React Snackbar component](docs/mui_docs/react-snackbar.md): Snackbars (also known as toasts) are used for brief notifications of processes that have been or will be performed.
- [React Speed Dial component](docs/mui_docs/react-speed-dial.md): When pressed, a floating action button can display three to six related actions in the form of a Speed Dial.
- [React Stack component](docs/mui_docs/react-stack.md): Stack is a container component for arranging elements vertically or horizontally.
- [React Stepper component](docs/mui_docs/react-stepper.md): Steppers convey progress through numbered steps. It provides a wizard-like workflow.
- [React Switch component](docs/mui_docs/react-switch.md): Switches toggle the state of a single setting on or off.
- [React Table component](docs/mui_docs/react-table.md): Tables display sets of data. They can be fully customized.
- [React Tabs component](docs/mui_docs/react-tabs.md): Tabs make it easy to explore and switch between different views.
- [React Text Field component](docs/mui_docs/react-text-field.md): Text Fields let users enter and edit text.
- [React Timeline component](docs/mui_docs/react-timeline.md): The timeline displays a list of events in chronological order.
- [React Tooltip component](docs/mui_docs/react-tooltip.md): Tooltips display informative text when users hover over, focus on, or tap an element.
- [React Transition component](docs/mui_docs/transitions.md): Transitions help to make a UI expressive and easy to use.
- [React Typography component](docs/mui_docs/react-typography.md): Use typography to present your design and content as clearly and efficiently as possible.
- [Textarea Autosize React component](docs/mui_docs/react-textarea-autosize.md): The Textarea Autosize component automatically adjusts its height to match the length of the content within.
- [Toggle Button React component](docs/mui_docs/react-toggle-button.md): A Toggle Button can be used to group related options.
